# coredump

My personal website and its associated blog. Written in [SvelteKit](https://kit.svelte.dev) and [TailwindCSS](https://tailwindcss.com/).

## Running locally

Install [bun](https://bun.sh), clone and `cd` into the repository, then run:

```bash
bun install

# for development:
bun dev

# for production:
bun build
bun start
```
